## Usage

---

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
$ source .env && forge script ./script/Crowdfunding.s.sol --rpc-url ${SEPOLIA_RPC_URL} --broadcast --verify
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
